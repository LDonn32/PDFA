# PDFA

Overview
This repository contains work for the Programming for Data Analytics (PFDA) module at ATU. It is part of the Higher Diploma in Science in Computing (Data Analytics) programme.

The repository is organized into three folders:

Contents
1. Assignments
This folder contains descriptions and completed assignments for the module.

2. My Work
This folder contains rough work, labs, lecture notes carried out during the semester. 

3. Project
This folder contains the  project for the module.
